# 依赖工具

  • tcpdump
  • netcat
  • wireshark

# 安卓上运行

tcpdump -w -  | nc -l -p 31337
1

# mac上运行

adb forward tcp:12345 tcp:31337
mkfifo /tmp/remote 
nc nc 127.0.0.1 12345 > /tmp/remote
wireshark -k -i /tmp/remote
1
2
3
4

nc确实很强大,很好用. 通过这种组合,就避免了安卓屏幕小的尴尬了.
